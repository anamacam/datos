### Tipos de datos
Los tipos de datos en Java son un formato de almacenamiento que pueden contener un valor, un rango o atributo específico.
## Datos Primitivos:
|Dato  |Valor |
|-------| ---:|
|byte   | 8 bits|
|short  | 16 bits| 
|int    | 32 bits |
|long   | 64 bits |
|float  | 32 bits |
|double | 64 bits |
|boolean| 1 bit |
|char   | 16 bits |

## ¿Qué es un objeto?
Un objeto puede ser una estructura de datos, una variable o una función.
